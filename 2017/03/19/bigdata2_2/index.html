有关编程：

  1. 编程中碰到了一些小问题。比如一开始没有计算数据集的正确范围（即最小值和最大值），也没有设置正确的步长，所以导致最后画出正态分布图很难看。
  2. 定义正态分布函数时，需要把函数定义在调用之前，否则会报错：NameError: name ‘normfun’ is not defined
  3. 在用jupyter调试时，有时候发现即使改了原文件，但用shift+enter执行时还是以前的结果。可以用“restart the kernel”菜单来重新执行看看。

数据分析：

  1. 大体上符合正态分布，即以149为均值中心，1.63为标准差的分布。
  2. 但从直方图上观察，并没有完美拟合正态分布，也可能是数据量过少。
  3. 用时146的概率看起来远大于用时152的概率，比较好理解，毕竟跑的慢的人还是占大多数。
