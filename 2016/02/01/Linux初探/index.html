Linux初探

Feb 1, 2016 | IT技术 | Hits

写Linux相关博文的动机在于自我再学习。去年考完RHCE以后就着手整理当时考试的资料以及平时工作中需要用到的知识点，所以能想到的一个比较好的方式是把这些东西写出来供自己参考，亦可以和朋友交流。在写作的过程中参考了一些相关的书籍以及当时培训的笔记，仅为参考学习素材，类似于笔记，不做商业用途使用。

Linux做为一种操作系统和Unix有着很深的渊源。Linux重新实现并优化了Unix内核，完全遵守POSIX标准，能够在不同的硬件平台运行，且兼容大多数Unix软件。那Linux到底是什么？Linux是一套操作系统，运行在硬件之上，包含内核和系统调用接口两层。不同于Windows，目前的Linux有很多发行版本，这些发行版本共享相同的内核源，但是有不同的辅助软件。各发行版本在其侧重点上有所不同，独立的Linux发行版有数百种之多，下面的表列举了目前比较流行的发行版本。

发行版本 说明
CentOS 模仿RHEL的免费发行版本
Redhat Enterprise Redhat公司的商业化发行版本
Debian 一种流行的非商业化发现版本
Linux Mint 基于Ubuntu，有优美的应用
Gentoo 用户用源代码自行编译和优化的发行版本
Oracle Enterprise Linux Oracle公司提供支持的RHEL版本
Slackware 古老而长寿的发行版本
Ubuntu Debian的精炼版本
Arch Linux 一种以轻量简洁为设计理念的Linux发行版

谈到Linux还有几个有关开源的项目和协议需要了解。

GNU: GNU’s Not UNIX，这个东西定义上看是一个无穷的循环。项目的本省是由Richard Mathew Stallman在1984年发起的，目的是创建一个自由，开放的UNIX操作系统，但最终的实现却是从把UNIX上的知名软件重写并开源出来开始的，比如GNU C Compiler（GCC，以前都是专利软件，需要付费的），GNU C Library等等。同时Stallman还成立了FSF（Free Software Foundation，自由软件基金会） 。为了避免GNU所开发的自由软件被其他人利用成为专利软件，Stallman还草拟了通用公共许可证（General Public License，GPL），并称为CopyLeft。

自由软件（Free Software）：”Free software” is a matter of liberty, not price. To understand the concept, you should think of “free speech”, not “free beer”. “Free software” refer to the users’ freedom to run, copy, distribute, study, change and improve the software.

当然，Linux最重要的贡献还在于芬兰小伙Torvalds的开创性的工作。1991年Linus Torvalds在BBS上贴了一则消息，宣称他用bash，gcc等工具写了一个小小的内核程序可以在386机器上运行了，版本号是0.02…..
